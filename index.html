Example Text Styles

H1: Main Page Heading

Display font: Baloo 2
Used to highlight title of the article.
Increase size if needed.

H2: (Sub)heading

Serif font: Merriweather
Used as a heading or subheading, more professional heading if required.

H3: Subheading

Handwriting font: Caveat
Used as a subheading only, a more personal connection to the reader. Not for use in every business.

P: Paragraph Text

Sans-serif font: Montserrat
Easy to read on a screen. Used for long articles.
